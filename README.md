# Excel-Project-Data-Set

Analyzed customer purchasing data using Microsoft Excel. Cleaned and transformed the dataset, created pivot table summaries, and built an interactive dashboard to provide insights into customer demographics and purchasing behavior.

Tasks Performed
1. Data Cleaning
Removed duplicate records.
Handled missing and inconsistent values.
Standardized data formats for analysis.
Verified data accuracy and completeness.
2. Data Analysis Using Pivot Tables

Created pivot tables to analyze key business metrics:

Average Income per Purchase
Examined the relationship between customer income levels and purchasing behavior.
Customer Commute Analysis
Analyzed customer distribution based on commute distance.
Customer Age Bracket Analysis
Grouped customers into age brackets and identified purchasing trends across age groups.
3. Interactive Dashboard Development

Built an Excel dashboard featuring:

Dynamic charts and visualizations.
Slicers for interactive filtering.
User-friendly layout for exploring customer demographics and sales trends.
Real-time updates based on selected filters.
Skills Used
Microsoft Excel
Data Cleaning
Pivot Tables
Data Analysis
Dashboard Design
Data Visualization
Slicers and Interactive Reporting
Outcome

Developed an interactive dashboard that transformed raw customer data into actionable insights, enabling easier analysis of customer demographics, commute patterns, and purchasing behavior.
